# Предварительные требования
Перед началом убедитесь, что у вас установлены:

- Node.js
- npm
- Git
- Expo

# Давайте начнем
1. Клонирование репозитория

```bash
git clone <your-repository-url>
cd your-app
```

2. Установка зависимостей

```bash
npm install
```
3. Запуск

```bash
npx expo start
```
После выполнения команды откроется интерфейс Expo Dev Tools с QR-кодом.

# Запуск на устройстве
Установите приложение Expo Go на свой телефон:

- Android Play Store
- iOS App Store

Отсканируйте QR-код из терминала

Приложение автоматически загрузится и запустится

# Настройка разрешений
Приложение автоматически запросит необходимые разрешения:

- Местоположение - для отслеживания позиции и уведомлений о приближении
- Камера/Галерея - для добавления изображений к меткам
- Уведомления - для показа оповещений о приближении к меткам

# Инструкция по тестированию

- Включите GPS
- Разрешите доступ к местоположению при запросе
- Проверьте, что синяя точка отображается на карте
- Перемещайтесь и убедитесь, что местоположение обновляется

# Сценарии тестирования местоположения

- Отказ в разрешении местоположения
- Отключение GPS

# Сценарии тестирования уведомлений

- Приближение к нескольким меткам одновременно
- Выход и повторный вход в радиус метки
- Удаление метки с активным уведомлением
- Перезапуск приложения с активными уведомлениями
